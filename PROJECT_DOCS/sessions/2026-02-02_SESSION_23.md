### ✅ Fix: PyQt6 ImportError (QShortcut)
- Datei: `pydaw/ui/notation/notation_palette.py`
- `QShortcut` wird in PyQt6 aus `PyQt6.QtGui` importiert (nicht aus `PyQt6.QtWidgets`).

### ✅ Perf: kleiner WAV/Render-Cache im Audio-Engine-Thread
- Datei: `pydaw/audio/audio_engine.py`
- Beim schnellen Stop/Play wurden WAVs (inkl. gerenderte MIDI-WAVs) immer wieder neu von Disk gelesen/decodiert.
- Neu: kleine LRU-Cache-Map im Engine-Thread (Standard: 16 Einträge) auf Basis `(path, samplerate)`.
- Cache speichert die **stereo float32** Daten **nach Resample** und **vor** Tempo-Stretch.

# Session Log — v0.0.20.251

## Task
Fix Pro Drum Machine Slot-FX rebuild loop / freeze when adding LADSPA effects.

## Root Cause
`SlotFxDeviceCard` connected `btn_power.toggled` before calling `setChecked()` in the constructor.
During every UI rebuild this emitted `toggled()`, which called the outer rebuild callback again.
That recreated the UI again and repeated indefinitely.

## Fix
Wrapped initial `setChecked()` with `QSignalBlocker`.

## Safety
UI-only initialization fix, no DSP architecture changes.

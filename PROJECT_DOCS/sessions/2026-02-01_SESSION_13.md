## Kontext / Problem
Beim Start (`python3 main.py`) trat ein **Fatal Python error: Aborted** auf.
Im Stacktrace war zu sehen:
- `QGraphicsView::paintEvent` → `QGraphicsItem::paint` → `pyqt6_err_print()` → `QMessageLogger::fatal()` → `SIGABRT`

Das ist ein klassisches PyQt6-Verhalten: **Uncaught Exceptions inside `QGraphicsItem.paint()` können den gesamten Prozess aborten**.

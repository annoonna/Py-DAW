- Service-seitig: `move_tab()` verschiebt im `_tabs` Array und korrigiert `_active_idx`

### Cross-Project Drop im Canvas
- Tab-Service wird per Setter (`set_tab_service()`) injiziert, nicht per Constructor
- Defensive: `_tab_service` kann `None` sein → graceful skip
- Logging bei erfolgreichen Cross-Project Copies

### Tab Navigation
- QShortcut statt QAction (keine Menü-Einträge nötig)
- Wrapping: Nach letztem Tab → erster Tab (Modulo)
- Guard: Nur aktiv wenn > 1 Tab offen

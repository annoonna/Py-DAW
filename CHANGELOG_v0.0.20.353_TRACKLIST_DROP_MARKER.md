# CHANGELOG v0.0.20.353 — TrackList Drop-Markierung beim Maus-Reorder

## Neu
- Linke Arranger-TrackList zeigt beim lokalen Maus-Reorder jetzt eine **sichtbare Drop-Markierung** an der Einfügeposition.
- Funktioniert für **einzelne Spuren**, **Mehrfachauswahl-Blöcke** und **Drop am Listenende**.
- Marker wird bei Drag Leave / Drop / Refresh wieder sauber entfernt.

## Sicherheit
- Kein Eingriff in Routing, Mixer, DSP, Playback oder Projekt-Audiodaten.
- Der bestehende **Cross-Project-Track-Drag** bleibt unangetastet.

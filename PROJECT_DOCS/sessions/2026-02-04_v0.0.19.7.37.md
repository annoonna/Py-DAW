# Session Log — v0.0.19.7.37

**Datum:** 2026-02-04

## Ziel
Hotfix für den Python-Logo-Button: Crash/Spam im `paintEvent` unter PyQt6 beheben.

## Problem (User-Report via GDB)
Beim Rendern des Python-Buttons kam wiederholt:
- `TypeError: QPainterPath.addEllipse(...): argument 1 has unexpected type 'QRect'`

Ursache: `self.rect().adjusted(...)` liefert `QRect`, aber `QPainterPath.addEllipse()` erwartet in PyQt6 einen `QRectF`.

## Fix
- Ellipse-Clip wird jetzt mit `r.toRectF()` gebaut.
- Ergebnis: keine Exceptions mehr im `paintEvent`, kein Traceback-Spam, UI bleibt stabil.

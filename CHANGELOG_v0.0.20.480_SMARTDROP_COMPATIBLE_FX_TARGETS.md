# v0.0.20.480 — SmartDrop: Kompatible FX-Ziele

Datum: 2026-03-15

## Neu

- ArrangerCanvas und linke TrackList akzeptieren jetzt **echte FX-SmartDrops** auf bereits kompatiblen bestehenden Zielen.
- `Note-FX` können auf bestehende **Instrument-Spuren** gedroppt werden.
- `Audio-FX` können auf bestehende **Instrument-/Audio-/Bus-/Gruppen-Spuren** gedroppt werden.
- Hover-/Tooltip-/Status-Texte unterscheiden jetzt auch für FX sauber zwischen **Einfügen** und **Nur Preview**.

## Safety

- Kein Audio→MIDI-Morphing.
- Kein Routing-Umbau.
- Inkompatible Ziele bleiben bewusst reine Preview.

## ZUSAMMENFASSUNG

Der Clip-Launcher „ZELLE“-Inspector war in der Praxis **zu schmal** und ließ sich nicht „wie in einer echten DAW“ aufziehen.
Ursache war eine harte Maximalbreite im Layout.

Fix: Inspector ist jetzt **frei skalierbar**, kann **ein-/ausgeklappt** werden und merkt sich **Breite + Sichtbarkeit**.

## WAS GENAU WURDE GEÄNDERT

### 1) Inspector ist wirklich resizable (keine harte Max-Breite)
- In `ClipLauncherPanel` wurde die harte Klammer entfernt/entschärft.
- Der User kann die ZELLE-Spalte jetzt per Splitter **beliebig breit ziehen**.

### 2) Einklappen/Ausklappen wie Bitwig/Ableton
- Neuer Header-Button (Pfeil) im Clip-Launcher Header.
  - Klick: Inspector **ausblenden** → Grid bekommt die volle Breite.
  - Klick erneut: Inspector **einblenden** → vorherige Breite wird wiederhergestellt.

### 3) Persistenz (QSettings)
- Neue Keys:
  - `ui/cliplauncher_inspector_visible`
  - `ui/cliplauncher_inspector_width`
- Dadurch bleibt der Zustand nach Neustart erhalten.

### 4) Scroll-Fallback
- Im Inspector ist die horizontale Scrollbar jetzt **AsNeeded** (nur falls jemand den Inspector extrem schmal macht).

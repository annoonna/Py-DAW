## 🔧 Änderungen

### 1) Keyboard Shortcuts
**File:** `pydaw/ui/notation/notation_view.py`

- `D` → Draw Tool  
- `S` → Select Tool  
- `E` → Erase Tool (als Left-Click Tool; Right-Click Erase bleibt erhalten)  
- `Ctrl+C` → Copy (Single-Note MVP)  
- `Ctrl+X` → Cut (Copy + Delete)  
- `Ctrl+V` → Paste (Single-Note MVP, Platzierung + Grid-Snap best-effort)  
- `Ctrl+Z` → Undo (ProjectService UndoStack)  
- `Del` / `Backspace` → Delete selected note (Undo-fähig)

### 2) Clipboard (MVP)
- Interne Note-Clipboard-Struktur (`_clipboard_note`)
- Paste-Position:
  - bei Auswahl: direkt hinter der ausgewählten Note
  - sonst: sequentiell weiter (last_paste_start + length)
  - sonst: Beat 0
- Snap: nutzt `project.snap_division` wenn vorhanden (best-effort)

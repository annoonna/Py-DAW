### 2) MainWindow wiring

### 3) Sampler Plugin komplett real im Repo
- `pydaw/plugins/sampler/`
  - `sampler_widget.py`: PyQt6 UI + Drag&Drop `.wav`, bindet `ProjectService.note_preview`
  - `sampler_engine.py`: monophoner Pull-Sampler (chromatisches Mapping `2**((note-root)/12)`)
  - `wav_io.py`: WAV loader (stdlib wave + numpy) + linear resample auf 48k
  - `dsp.py`: placeholder (future FX)
  - `__init__.py`: export `SamplerWidget`

### 4) PianoRoll → Preview Note
- `pydaw/ui/pianoroll_canvas.py`
  - nach `commit_midi_notes_edit(..., "Add Note")` wird `project.preview_note(pitch, 100, 140)` ausgelöst.

### 5) AudioEngine Pull-Sources + Preview Output
- `pydaw/audio/audio_engine.py`
  - neue API: `register_pull_source(name, fn)` / `unregister_pull_source(name)`
  - `ensure_preview_output()`:
    - JACK: setzt DSP callback (mischt Pull-Sources) auch ohne Arrangement
    - sounddevice: startet Silence-Stream und mischt Pull-Sources
  - sounddevice-silence: mischt Pull-Sources in Callback
  - sounddevice-arrangement: mischt Pull-Sources zusätzlich zum Clip-Mix
  - soft clip via `np.clip(..., out=...)` (kein Extra-Array)

## ALGORITHMUS

Die Chain-Layout-Reihenfolge ist:
```
[Note-FX Card 0] [Note-FX Card 1] ... [Instrument Anchor] [Audio-FX Card 0] [Audio-FX Card 1] ... [Stretch]
```

Beim DragMove:
1. Event-Position wird via `globalPosition` → `mapFromGlobal` auf chain_host-Koordinaten umgerechnet
2. `_get_chain_card_positions(kind)` sammelt (x, width) aller Cards der passenden Zone
3. Cursor-X wird mit Card-Centers verglichen → Insert-Index berechnet
4. Drop-Indicator wird an der berechneten Gap-Position angezeigt

Beim Drop:
1. `_drop_insert_kind` + `_drop_insert_index` werden gelesen
2. Indicator wird versteckt
3. `add_*_fx_to_track(..., insert_index=idx)` nutzt `list.insert()` statt `append()`
